# TYPO3 extension base_svg

## What is does

This extension brings the "SVG file parsing / rendering library" Dompdf PHP classes "phenx/php-svg-lib" to TYPO3.

## What is the origin

See https://github.com/dompdf/php-svg-lib/tree/0.5.0
